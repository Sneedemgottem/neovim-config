## Required Dependencies for LSP and Mason:
1. Required for Mason
..* unzip
..* wget
..* curl
..* gzip
..* tar
..* bash
..* sh
..* nvim version >= 0.7.0

2. Required for Tree Sitter
..* build tools (gcc, cc, clang)
..* node (optional)
..* git

3. Required for Telescope
..* Plenary
..* nvim-treesitter
..* rg (ripgrep)
..* fd
